1. Difference between Map, Filter ForEach, Find ?

Answer: forEach() is execute same code on every element in an array but does not return. map() is execute like as forEach() but it return array. filter() is used for check every element according to condition and returns a new array which has all element those satisfy with condition. find() is worked as filter but in returns gave first value of this array which satisfy the conditions

2. Difference between Local storage, Session storage, Cookies ?

Answer: Local storage: The data which stored in local storage is not deleted when the browser is closed. So really use things for access data later. Session storage: The data which stored only for session and deleted data when browser is closed. It helps for improve web application performance. Cookies: the data which saved will be expire at a certain time. The data will sent a request client to server.

3. What is event loop and how it works ?

Answer: The event loop is a process which not pushing callbacks from the Task Queue to the Call Stack, until the Call Stack to be clear. When stack is clear,the event loop triggers and checks the Task Queue for available callbacks.And When stack is full wait in the callback queue until clear stack. This things repeat.
