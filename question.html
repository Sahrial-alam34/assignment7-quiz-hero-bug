Question And Answer!!

1. What is DOM ? What is the purpose of DOM ?

Answer: The HTML DOM is an object model that used in HTML section as programming interface. It looks like a logical tree. DOM represents a documents as nodes and object. When a web browser parses an HTML document,it builds a DOM tree and then uses it to display the document.

2. How will you select HTML elements using DOM? Name the DOM methods ?

Answer: To select HTML elements within a document using JavaScript is used in DOM selectors. There five ways for select: a. getElementsByTagName(). b. getElementsByClassName(). c. getElementById(). d. querySelector(). e. querySelectorAll().

3. What is event bubble ?

Answer: When a an event is in an element inside another element. Both elements bounded for that event. It is a process which starts with the element that triggered the event and then bubbles up to the containing elements in the down or hierarchy.

4. What is a callback function and why will you use it?

Answer: A function which used for passed as an arguments to another function. It runs after another function has finished. It passed another function parameters as value not function type. When we call a function, we can control this function. A callback's primary purpose is to execute code in response to an event.
